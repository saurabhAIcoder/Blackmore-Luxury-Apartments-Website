# Blackmore Luxury Apartments Website

## 📌 Project Overview

This project is a **Luxury Apartment Landing Page** built using **HTML and CSS**.
It showcases a modern apartment website layout with sections like hero banner, location information, apartment details, gallery, testimonials, and booking call-to-action.

## 🛠 Technologies Used

* HTML5
* CSS3
* Flexbox Layout

## ✨ Features

* Modern landing page design
* Navigation bar with menu links
* Hero section with heading and CTA button
* Location and apartment information sections
* Image gallery
* Customer testimonial section
* Booking call-to-action
* Footer with navigation links

## 🎯 Purpose

The purpose of this project is to practice **HTML structure, CSS styling, and building a clean landing page layout** for a real estate or apartment rental website.

## 🚀 How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open **travel.html** in your web browser.
